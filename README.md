# Dorian Apps Website

Multipage website for `dorianapps.com`, the software company of Nikolcho Velkovski.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- GSAP ScrollTrigger
- Lenis smooth scroll
- SplitType text reveals
- Three.js hero centerpiece

## Correct Content Model

Products:
- DocVault
- CodexPilot
- FENIX Construction Tracker

Website work:
- Fenix Plus SA, a company website built by Nikolcho Velkovski at `fenixplus.ch`

FENIX Construction Tracker owns the login and live-map screenshots. Fenix Plus SA is separate website work and must not be represented as the tracker app.

## Local Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

## Verification

```bash
npm test
npm run lint
npm run build
```

For visual QA, run the production server and inspect the home page, product routes, Fenix Plus SA work page, mobile navigation, hero video scrub, and Three.js canvas.
